Para identificar con mayor precisin aquellos datos atpicos se utilizar el Mtodo de Tukey, el cual utiliza el rango intercuartil para filtrar nmeros muy grandes o muy pequeos. Utilizando las siguientes frmulas para ello: [2]

  • Valores atpicos bajos = $Q1 1,5 (Q3 Q1) = Q1 1,5 (IQR)$

  • Valores atpicos altos = $Q3 + 1,5(Q3 Q1) = Q3 + 1,5(IQR) $

Los valores que se encuentran por debajo de Q1 1.5IQR o por encima de Q3 + 1.5IQR se consideran atpicos (outliers). [2]

Este procedimiento se realizar calculando los cuartiles (0.25 y 0.75) con quantile(), obteniendo el IQR y estableciendo los lmites inferior y superior. Para posteriormente filtrar las observaciones fuera de los lmites establecidos.

In[41]:
cols = ['Edad','Faltas','G1','G2','G3']

for col in cols:
    # Cuartiles
    Q1 = data[col].quantile(0.25)
    Q3 = data[col].quantile(0.75)

    # Rango intercuartil
    IQR = Q3 - Q1

    # Lmites
    lower = Q1 - 1.5 * IQR
    upper = Q3 + 1.5 * IQR

    # Filtrar valores
    outliers = data[(data[col] < lower) | (data[col] > upper)][col]

    # Resultados
    print(f"Outliers en {col}:")
    print(outliers.values)

Anlisis de relaciones entre variables

En esta seccin se analizarn las relaciones entre las variables explicativas con el propsito de identificar posibles problemas de colinealidad o redundancia de informacin. Siendo esencial para garantizar que el modelo sea entrenado con variables que brinden un valor independiente, evitando sesgos debido a informacin duplicada.

Para ello, en cuanto a las variables cuantitativas (Edad, Faltas, G1, G2 y G3) se utilizar el Coeficiente de Correlacin de Pearson, dado que las variables consideradas son numricas y se busca identificar relaciones lineales que puedan generar colinealidad.

Este coeficiente mide la direccin de la relacin lineal entre dos variables continuas, con valores que oscilan entre -1 y +1. [4]

Un valor cercano a 1 indica una correlacin positiva fuerte, mientras que uno cercano a -1 refleja una correlacin negativa fuerte; valores prximos a 0 sugieren ausencia de relacin lineal. [4]

Cabe destacar que, que Pearson identifica relaciones lineales, por lo que, en caso de que los datos cuenten con una tendencia no lineal, el coeficiente puede parecer bajo aun cuando exista una asociacin significativa. [4]

De manera que, se seleccionarn las variables numricas con la funcin data.[nums_vars].corr() representndose en un mapa de calor mediante sns.heatmap. Agregando detalles al grfico con plt.figure y plt.title, para finalmente mostrarse con plt.show().

En cuanto a Edad y Faltas, no muestran correlaciones fuertes, aportando as informacin independiente.

En cuanto a las variables cualitativas, Sexo e Internet, no se espera una relacin evidente entre dichas variables. No obstante, para confirmar que no existe asociacin estadsticamente significativa se realizar una prueba Chi-cuadrado.

El test Chicuadrado es una prueba estadstica para datos categricos que permite evaluar si dos variables cualitativas estn relacionadas o si son independientes, calculndose mediante la siguiente frmula: [5]

$\chi^2 = \sum \frac{(O-E)^2}{E}$

Donde, $O$ son las frecuencias observadas y $E$ las esperadas. De manera que, si el pvalor 0.05, no hay evidencia de asociacin.

Para ello se utilizarn las funciones pd.crosstab para la tabla de contingencia y posteriormente la prueba Chi-cuadrado chi2_contingency de la librera spicy.stats.

In[49]:
import pandas as pd
import scipy.stats as stats

# Sexo vs Internet
tabla = pd.crosstab(data['Sexo_M'], data['Internet_yes'])
chi2, p, dof, expected = stats.chi2_contingency(tabla)

Seleccin de caractersticas

En modelos de regresin lineal mltiple, no todas las variables contribuyen de manera relevante al resultado. Donde, al incluir demasiadas variables se puede general colinealidad y dificultar interpretacin. Por lo cual, se utilizarn mtodos de seleccin de caractersticas que ayudan a simplificar el modelo y hacer nfasis en los factores ms importantes.

El mtodo de eliminacin hacia atrs (Backward Elimination) es una tcnica de seleccin de variables, que comienza con un modelo completo (todas las variables incluidas) y elimina de manera iterativa aquellas que no son estadsticamente significativas. [7]

Para ello se siguen los siguientes pasos: [7]

  1. Definir un nivel de significancia (p-value < 0.05).
  2. Ajustar el modelo con todas las variables.
  3. Identificar la variable con el mayor pvalue. Si ese pvalue es mayor al nivel de significancia, se elimina la variable.
  4. Recalcular el modelo y repetir el proceso hasta que todas las variables restantes tengan pvalues menores al nivel de significancia.

Asegurando que el modelo final conserve nicamente las variables con evidencia estadstica.

Para el anlisis del estudio contamos con mltiples variables que podran influir en el rendimiento acadmico. Sin embargo, sabemos que algunas de ellas, como G1 y G2, presentan una correlacin muy importante con la variable dependiente (G3). Por esta razn se considerarn dos modelos distintos.

El cdigo que se utilizar para ello, utiliza la librera statsmodels para ajustar el modelo de regresin lineal mltiple con la variable dependiente G3.

Primeramente, se definirn todas las variables explicativas, tomando en cuenta la constante con la funcin sm.add_constant para incluir l intercepto. Luego, con sm.OLS(y, X).fit() se estimar el modelo, obteniendo mtricas R y pvalues.

As, a travs de un ciclo while, se identificar la variable con el mayor pvalue usando pvalues.idxmax().Donde, si supera el nivel de significancia de 0.05, se elimina de la lista con variables.remove().

El proceso ser repetido hasta que solo permanezcan las variables significativas, definiendo al final el conjunto de caractersticas seleccionadas.

  • Se espera que el porcentaje explicativo ($R^2$) sea alto, ya que estas notas se correlacionan fuertemente con G3.

import statsmodels.api as sm

y = data['G3']

# Variables consideradas
variables = ['Reprobadas','Edad','Faltas','Sexo_M',
             'Horas_2','Horas_3','Horas_4','Internet_yes',
             'G1','G2']

while True:
    X = sm.add_constant(data[variables])
    modelo = sm.OLS(y, X).fit()
    print("\nVariables:", variables)
    print("R:", round(modelo.rsquared, 3))
    print("p-values:\n", modelo.pvalues)

    # Condiciones para eliminar variable con mayor p-value
    pvals = modelo.pvalues.drop('const')
    peor_var = pvals.idxmax()
    if pvals[peor_var] > 0.05:
        variables.remove(peor_var)
        print(f"Eliminada: {peor_var}")
    else:
        break

print("\nModelo final:")
print(variables)
