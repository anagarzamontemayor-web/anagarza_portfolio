#### 2.3 Selección de variables

Por su parte, los modelos que se implementarán presentan diferentes sensibilidades frente a la relevancia y redundancia de las variables, optando por realizar una selección de variables específica para cada tipo de modelo.

En el caso de los modelos lineales, como la Regresión logística, Support Vector Machine (SVM) y Linear Discriminant Analysis (LDA), se priorizará la eliminación de variables altamente correlacionadas. En el caso de la red neuronal, se mantendrán las variables relevantes evitando un variables de ruido que puedan dificultar el aprendizaje.

Por otro lado, modelos como Random Forest y Boosting, son menos sensibles a la presencia de variables irrelevantes o correlacionadas, por lo que se utilizará un conjunto más amplio de características sin llegar a afectar significativamente su desempeño.

La selección de variables se realizará únicamente sobre el conjunto de entrenamiento, buscando así que no exista fuga de información hacia el conjunto de prueba.

#### 2.4 Modelo basado en Redes Neuronales

Finalmente, se implementará una Red Neuronal sencilla. Este modelo se encuentra compuesto de capas de nodos: una capa de entreda, capas ocultas y una capa de salida. Donde, cada nodo se conecta con otros, contando con un peso y umbral asociado. De manera que, si la salida de un nodo supera el umbral, se activa y envía datos a la siguiente capa. [8]

De manera que, buscamos que la red aprenda las relaciones entre los indicadores socieconómicos y la pobreza en México.
